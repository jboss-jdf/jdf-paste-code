Useful Information

Paste Code:

  • Enter your nick - this is optional, if omitted, your paste will be anonymous
  • Choose the language of your code - this allows syntax highlighting
  • Choose if this code is private - when this option is chosen the application will generate a hash value of your code and will provide you with a unique link (something like http://localhost:8080/weld-pastecode/582ad4). Furthemore your code will not be listed in the right panel and cannot be found through the search.
  • Enter your code - you can also add a note
  • Submit the form!

Search Pastes:

  • Go to a history page - now you can see all the posts with pagination
  • You can narrow down the pastes by using the search form

Weld Features Covered:

  • Injection into POJO, EJBs and Servlets
  • Use of WAR packaging
  • Producers
  • JPA access
  • Decorators


Enjoy working with Weld!
